# Healthcare-Analysis
This project aimed to uncover insights from Healthcare data through advanced data analysis techniques, leveraging Excel functions and SQL queries, and visualizing key findings through interactive dashboards developed in Power BI, Excel and Tableau.

### Project Targets:
* Develop a Healthcare analytics dashboard to track and visualize key patient metrics including Total patients, visits, and doctor counts.
* Develop an integrated data model connecting Patient, Doctor, Visit, and Treatment datasets for a holistic view of Healthcare operations.
* Create a detailed visualizations for patient demographics, highlighting Average patient age and Geographical distribution across States like California and Florida.
* Generate an insights into Top diagnosed conditions, identifying Migraine, Hypertension, and Asthma as the most prevalent health issues.
* Develop a tracking system for Lab test results, categorizing outcomes into Normal, Abnormal, and Pending statuses for over 10,000 records.
* Create a Performance monitoring module for Doctor workload and Hospital affiliation to optimize Healthcare resource allocation.
* Generate Average treatment cost across various categories including Medication, Surgery, Physical Therapy, and Vaccinations.
* Develop a follow-up rate tracking mechanism to monitor patient continuity of care and identify areas for improved engagement.
* Create specialized reports on Lab test accuracy and reference ranges to ensure High Standards of diagnostic reliability.
* Generate automated summaries of Patient medical history, Allergies, and Chronic conditions to streamline clinical decision-making.

### Tools Used:
* **Excel** : **Leveraged Excel's advanced functionality, data modeling, and data visualization.**

* **Power Bi** : **Built dynamic dashboards to facilitate exploratory data analysis of Healthcare data.**

* **Tableau** : **Leveraged Tableau's data visualization capabilities to analyze and interpret Healthcare data.**

* **MySql** : **Applied SQL expertise to manage and analyze data.**

### Project Rationale :
This initiative aimed to showcase technical expertise in data analysis and visualization, employing Excel, SQL, Power BI, and Tableau to analyze Healthcare data and uncover key trends, facilitating data-driven decision-making.

### Outcome :
The project's outcome was a detailed set of healthcare analytics reports and interactive dashboards, equipping medical administrators with critical insights to optimize hospital operations, improve patient follow-up care, and enhance clinical diagnostic accuracy.
